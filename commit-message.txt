feat: Integrate Extension Marketplace with Open WebUI

This commit implements the integration between Open WebUI and the extension marketplace:

- Create a new marketplace module with API client for fetching extensions
- Add marketplace configuration with endpoints for the extension registry
- Implement UI components for browsing and installing marketplace extensions:
  - MarketplaceTab: Main component for marketplace browsing
  - MarketplaceCard: Component to display extension information
  - MarketplaceFeatured: Component for featured extensions
  - ExtensionDetails: Component for detailed extension information
- Update ExtensionManager to include a marketplace tab
- Update extensions index to export the marketplace module

The integration allows users to:
- Browse extensions in the marketplace
- View extension details including documentation
- Install extensions directly from the marketplace
- Filter extensions by category and type
- Search for extensions

This enhances the user experience by providing a centralized repository of
extensions that can be easily discovered and installed.
