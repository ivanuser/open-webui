feat(mcp): Add comprehensive Model Context Protocol support

This commit adds a new MCP (Model Context Protocol) integration for Open WebUI, allowing Ollama models to communicate with MCP servers for enhanced capabilities:

- Server management: Add, start, stop, and monitor MCP servers from the UI
- Tool integration: Enable Ollama models to use MCP tools
- Admin interface: View server status and logs
- CLI: Manage MCP servers from the command line
- Documentation: Add detailed documentation for MCP features

Key components:
- Python backend with server manager and API endpoints
- Frontend UI components in the admin panel
- Chat UI integration for tool usage
- Bridge between Ollama and MCP servers
- Multiple transport types support (stdio, sse)

Supported server types:
- Filesystem: Access files and directories
- Memory: Knowledge graph-based persistent memory
- Brave Search: Web search capabilities
- GitHub: Repository management

This implementation follows best practices from successful MCP projects and is designed to be extensible for future MCP server types.
