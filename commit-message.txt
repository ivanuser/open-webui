feat(mcp): Implement comprehensive MCP integration

This commit adds a complete implementation of the Model Context Protocol (MCP) to Open WebUI:

- Add MCP server management system for installing, starting, and stopping MCP servers
- Implement backend API for MCP server and tool management
- Create frontend UI components for managing MCP servers in the admin panel
- Integrate MCP with Ollama models to enable tool usage
- Support multiple server types (filesystem, memory, web search, etc.)
- Add Python utilities for server management
- Create documentation for using MCP in Open WebUI

The implementation follows the standard MCP specification and works with both
JavaScript and Python MCP servers. Users can now extend their AI models with
tool capabilities through a standardized protocol.
