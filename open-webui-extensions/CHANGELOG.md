# Changelog

All notable changes to the Open WebUI Extension System will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-03-15

### Added

- Initial release of the Open WebUI Extension System
- Extension Manager with backend and frontend components
- Extension Framework with base classes for different extension types
- Extension Registry for discovering and loading extensions
- REST API endpoints for managing extensions
- Hook system for extensions to integrate with Open WebUI
- Example extension demonstrating the system
- Detailed documentation for creating and managing extensions
- Installation scripts for easy deployment
- Integration with the existing Open WebUI extension system

### Fixed

- Build error in `src/lib/apis/openai/index.js` by explicitly re-exporting `chatCompletion`
